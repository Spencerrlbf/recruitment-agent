# `reports/retrieval_eval/`

Use this directory for generated retrieval-evaluation artifacts.

Typical contents:
- offline evaluation summaries
- score breakdowns and cohort comparisons
- exported candidate and job samples used for retrieval tuning

Naming convention:
- `YYYYMMDD__retrieval_eval_<cohort>.<ext>`
- examples: `20260414__retrieval_eval_baseline.md`, `20260414__retrieval_eval_gold_set.json`

Generated files in this directory are ignored by git. Keep only the scaffold and documentation tracked.
